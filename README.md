# Aplikasikuh
Aplikasikuh is a QML app that can help with daily activities. With its modularity, Aplikasikuh's features can be personalized to user's liking, like hobbies and even work. 
Aplikasikuh is still only available in Windows, but in the future work, Aplikasikuh can be accessed in Linux, IOS, and Android.
Aplikasikuh utilizes QML for its UI, with C++ as the backend and SQLite for the database. Aplikasikuh also use Github REST API to store the database online.

### This app is still underdevelopment. Releases will be uploaded if the project is done, along with the plugins in the packages.
